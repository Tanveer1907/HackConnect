# Hackathon + Internship Collaboration Platform

## 📌 Project Overview
A verified platform for college students to:
- Discover upcoming hackathons
- Form intelligent cross-college teams
- Showcase skills and achievements
- Get internship recommendations based on skills and hackathon performance

---

## 🎯 Problem Statement
Students struggle to:
- Find reliable hackathon teammates
- Discover relevant hackathons
- Convert hackathon experience into internships

This platform solves these problems through verified login, skill-based collaboration, and intelligent filtering.

---

## 🚀 Core Features
- College email based authentication
- Student profile with skills & achievements
- Hackathon discovery (categorized)
- Team formation system
- Internship recommendations
- Secure backend with JWT authentication

---

## 🛠 Tech Stack
Frontend:
- React.js
- Tailwind CSS

Backend:
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- bcrypt

Deployment:
- Vercel (Frontend)
- Render / AWS (Backend)
